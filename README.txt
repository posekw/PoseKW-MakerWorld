🚀 خطوات التثبيت (سهلة جداً):
───────────────────────────────────────────────────────────────

الخطوة 1: فك الضغط
──────────────────
1. انقر كليك يمين على الملف: PoseKW-MakerWorld-v7.4-FINAL.zip
2. اختر: "Extract All" أو "فك الضغط"
3. سيظهر 3 ملفات

الخطوة 2: رفع الملفات
──────────────────────
1. افتح: FTP أو cPanel File Manager
2. اذهب لمجلد: wp-content/plugins/
3. أنشئ مجلد جديد باسم: posekw-makerworld
4. ارفع الملفات الثلاث فيه

الخطوة 3: تفعيل الإضافة
──────────────────────
1. اذهب لـ: WordPress Admin
2. اختر: Plugins (الإضافات)
3. ابحث عن: PoseKW MakerWorld
4. اضغط: Activate (تفعيل)

الخطوة 4: مسح الـ Cache
──────────────────────
1. اضغط: Ctrl+Shift+Delete
2. اختر: Delete All (حذف الكل)
3. اضغط: Clear (مسح)

الخطوة 5: استخدم الإضافة
──────────────────────
في أي صفحة اكتب:
[posekw_makerworld_search]
